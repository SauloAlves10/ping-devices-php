# Ping Devices PHP

Desenvolvido com uma das linguangens de programação mais utilizadas e banco de Dados MySql, Ping Devices PHP é uma ferramenta simples e de fácil uso que te ajudará a realizar testes de ping em diversos devices em sequência, gerando uma lista com a nomenclatura dos itens, números de IPs, resultados dos testes de ping e se os itens se encontram disponíveis ou não.
